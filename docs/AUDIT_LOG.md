| O-001 | Auth | High | Legacy NextAuth credentials provider may still be reachable. README under `docs/_reference/README.md` describes a credentials login with seeded `admin@acoms.local`; live `src/middleware.ts` is simplified to "no more 2FA logic, login pages, or TwoFactorGuard". ## How to update this file

- **Found something?** Add a row immediately. ID format `O-NNN`.
- **Fixing something?** Mark `Fixed`, add the date and commit SHA, leave the row in place.
- **Severity guidance:** Critical = production-breaking; High = data-integrity / security; Medium = correctness / hygiene; Low = drift / cleanup; Info = noted-but-no-action.
